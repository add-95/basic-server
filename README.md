# Basic-Server
 Getting started with the creating first basic server concept


#
> ## To Run this code in your device Install these:-
> * ```npm install express```
> * ```npm install ejs```

#
> ## _To run the Server :_
> * ```npm start```
#

> ## **URLs**

> * _Home Page URL :_
```localhost:8000```
> * _Users Profile Page URL :_
```localhost:8000/users/profile```
